## 🎨 Personalización
El proyecto utiliza variables CSS para los colores principales y efectos. Puedes modificar estos valores en:

```css
/* src/styles/globals.css */
:root {
  --neon-blue: #00f7ff;
  --neon-orange: #ff4d00;
  --neon-purple: #9900ff;
}
```
